## 功能说明<a name="section20806203412478"></a>

关闭对应域打点的采集。

可以通过多次调用接口来关闭多个域的采集。默认所有域的采集均已开启。

## 函数原型<a name="section1121883194711"></a>

```cpp
msptiResult msptiActivityDisableMarkerDomain(const char* name)
```
